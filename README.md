# homebrew-vizhiocr

Homebrew tap for [VizhiOCR](https://github.com/purple-glass-dev/vizhi-ocr) — an on-device OCR menu bar app for macOS.

## Install

```sh
brew install --cask purple-glass-dev/vizhiocr/vizhi-ocr
```

Or tap first, then install:

```sh
brew tap purple-glass-dev/vizhiocr
brew install --cask vizhi-ocr
```

## Upgrade

```sh
brew upgrade --cask vizhi-ocr
```

_The cask in `Casks/` is generated by `scripts/update-cask.sh` in the app repo on each
notarized release — edit it there, not here._
